🚀 Getting Started

1
Open the Dashboard
Click the shortcut created on your Desktop or Start Menu. The app opens in your default browser.
2
Choose Your Tweaks
Browse the tabs and toggle the tweaks you want to apply. Each one has a clear description and risk level.
3
Run or Export
Click "Run Now" to apply immediately, or "Export .ps1" to review the script first.
4
Run as Administrator
To apply system-level tweaks, run the exported script as Administrator:
Right-click → Run with PowerShell → Yes (UAC)
